
**测试账号**:
```

密码: 任意
```

### 切换真实后端（可选）

如需使用真实数据和AI识别：

1. **安装环境**
   ```bash
   brew install openjdk@17 maven
   ```

2. **启动后端**（IntelliJ IDEA）
   - 打开 `backend` 文件夹
   - 点击运行按钮

3. **修改前端配置**
   ```javascript
   // frontend/src/utils/request.js
   const USE_MOCK = false;  // 改为false
   ```

4. **重启前端**
   ```bash
   npm run dev
   ```
