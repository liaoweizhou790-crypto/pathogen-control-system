## 🚀 启动命令

### 1. 数据库初始化
```bash
# 创建数据库和表
mysql -u root -p < database/init/01_create_database.sql

# 插入测试数据
mysql -u root -p < database/init/02_test_data.sql
```

### 2. 后端启动
```bash
cd vector-monitoring-platform/backend
mvn clean install
mvn spring-boot:run

# API地址: http://localhost:8080/api
# 接口文档: http://localhost:8080/api/doc.html
```

### 3. 前端启动
```bash
cd vector-monitoring-platform/frontend
npm install
npm run dev

# 访问地址: http://localhost:3000
```

## 🎯 核心功能演示

### 1. 登录流程
```
用户输入用户名密码 → 后端验证 → 生成JWT Token → 前端存储 → 跳转首页
```

### 2. AI识别流程
```
用户拍照上传 → 后端接收 → Base64编码 → 调用Kimi API → 返回JSON识别结果 → 前端填充表单
```

### 3. 数据录入流程
```
选择监测类型 → 填写表单（或使用AI识别）→ 自动计算密度 → 提交到后端 → 保存数据库
```

### 4. 报告生成流程
```
选择报告类型（月/季/年）→ 后端查询数据 → 统计分析 → 生成Markdown报告 → 返回给前端
```

### 5. 数据导出流程
```
设置查询条件 → 调用导出API → 后端查询数据 → EasyExcel生成Excel → 浏览器下载
```
