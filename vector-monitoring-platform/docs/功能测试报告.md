## 常见问题排查

### 问题1: 页面空白或加载不出来
**解决方案**:
```bash
# 1. 检查端口占用
lsof -ti:3003 | xargs kill -9

# 2. 清除缓存重新启动
cd /Users/liaoweizhou/.openclaw/workspace/vector-monitoring-platform/frontend
rm -rf node_modules/.vite
cnpm run dev
```

### 问题2: 登录后跳转到空白页
**原因**: 可能是路由守卫问题
**解决方案**: 刷新页面或重新登录

### 问题3: 图表不显示
**原因**: ECharts库加载问题
**解决方案**: 刷新页面，检查网络

### 问题4: 数据提交失败
**原因**: Mock模式正常工作，只是提示模拟
**说明**: 在Mock模式下，数据不会真实保存，但会显示成功提示
