## 📱 功能导航

### 1️⃣ 数据看板
**路径**: 首页（登录后自动进入）

**功能**:
- 📊 查看4类病媒密度统计（蚊、蝇、蟑、鼠）
- 📈 查看季节消长趋势图
- 🗺️ 查看城区分布对比
- ⚠️ 查看风险预警
- 📅 切换时间周期（本月/本季/本年）

**操作**:
- 点击顶部"本月/本季/本年"按钮切换时间

---

### 2️⃣ 数据录入
**路径**: 侧边栏 → 数据录入

**功能**:
- 📝 支持8种监测类型：
  - 🦟 蚊密度监测（诱蚊灯法）
  - 🪰 蝇密度监测（捕蝇笼法）
  - 🪳 蟑密度监测（粘蟑法）
  - 🐭 鼠密度监测（粘捕法）
  - 🏕️ 布雷图指数
  - 🏕️ 双层叠帐法
  - 🎯 诱卵器监测
  - 🪲 蜱虫监测
- 🤖 AI辅助识别（拍照自动填表）
- 🎤 语音录入（预留功能）
- 📋 复制上条记录
- 🧮 自动计算密度值

**操作步骤**:
1. 选择监测类型
2. 填写基本信息（日期、区县、监测点、环境类型）
3. 填写监测数据（布放数、捕获数等）
4. 系统自动计算密度值
5. 点击"提交数据"

**AI识别使用**:
1. 点击"📷 拍照识别记录表"
2. 选择图片上传
3. 等待识别结果（模拟2秒）
4. 查看置信度和识别数据
5. 点击"应用识别结果"

## 🔧 常见问题

### Q1: 页面打不开或空白
**原因**: 服务未启动或端口冲突
**解决**:
```bash
# 1. 检查服务状态
curl http://localhost:3003

# 2. 如果无响应，重新启动
cd /Users/liaoweizhou/.openclaw/workspace/vector-monitoring-platform/frontend
pkill -f vite
npm run dev
```

### Q2: 登录后跳转到空白页
**原因**: 可能是路由加载问题
**解决**:
1. 刷新页面（F5）
2. 清除浏览器缓存（Ctrl+Shift+R）
3. 重新登录

### Q3: 图表不显示
**原因**: ECharts库加载延迟
**解决**:
1. 等待几秒钟让图表加载
2. 刷新页面
3. 检查浏览器控制台是否有错误

### Q4: 数据提交后不见了
**说明**: Mock模式下数据不持久化，刷新页面会重置
**解决**: 这是正常现象，仅用于演示。如需真实保存，需要连接后端服务。

### Q5: AI识别一直加载中
**原因**: 模拟延迟或网络问题
**解决**:
1. 等待2-3秒
2. 如果一直加载，刷新页面重试
3. 检查控制台是否有错误信息

## 📊 数据说明

### Mock模式
当前系统运行在 **Mock模式** 下：
- ✅ 所有功能可正常使用
- ✅ 数据为模拟数据
- ⚠️ 数据不会真实保存（刷新后重置）
- ⚠️ AI识别为模拟效果

### 切换到真实后端
如需使用真实数据和AI识别：
1. 安装 IntelliJ IDEA
2. 打开 `backend` 文件夹
3. 点击运行按钮启动后端
4. 修改 `frontend/src/utils/request.js` 中的 `USE_MOCK = false`
5. 重启前端
