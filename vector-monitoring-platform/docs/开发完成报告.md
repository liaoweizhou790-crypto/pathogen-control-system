## 🚀 启动说明

### 1. 数据库初始化
```bash
mysql -u root -p < database/init/01_create_database.sql
```

### 2. 后端启动
```bash
cd backend
mvn clean install
mvn spring-boot:run
# API地址: http://localhost:8080/api
```

### 3. 前端启动
```bash
cd frontend
npm install
npm run dev
# 访问: http://localhost:3000
```

## ⚙️ 配置说明

### 后端配置 (application.yml)
```yaml
# 数据库
spring.datasource:
  url: jdbc:mysql://localhost:3306/vector_monitoring
  username: root
  password: 123456

# JWT
jwt:
  secret: your-secret-key-here-change-in-production
  expiration: 86400000

# Kimi API (从 https://platform.moonshot.cn 获取)
kimi.api:
  key: your-kimi-api-key
  url: https://api.moonshot.cn/v1/chat/completions
  model: moonshot-v1-8k-vision-preview
```

## 📱 API接口清单

### 用户认证
- `POST /api/user/login` - 用户登录
- `GET /api/user/info` - 获取用户信息

### 数据看板
- `GET /api/dashboard/stats` - 统计数据
- `GET /api/dashboard/trend` - 趋势数据
- `GET /api/dashboard/district-comparison` - 区县对比

### 数据录入
- `POST /api/monitoring` - 新增监测数据
- `PUT /api/monitoring/{id}` - 更新数据
- `DELETE /api/monitoring/{id}` - 删除数据
- `GET /api/monitoring/{id}` - 获取详情
- `GET /api/monitoring/list` - 获取列表

### AI识别
- `POST /api/ai/recognize` - 图片识别

### 报告中心
- `POST /api/reports/monthly` - 生成月报
- `POST /api/reports/quarterly` - 生成季报
- `POST /api/reports/annual` - 生成年报
- `POST /api/reports/custom` - 生成自定义报告
- `GET /api/reports/export` - 导出Excel

## 🖼️ 功能演示

### 1. 登录流程
```
用户输入用户名密码 → 后端JWT验证 → Token存储 → 跳转首页
```

### 2. AI识别流程
```
用户上传图片 → 后端接收 → Kimi Vision API识别 → 返回JSON → 前端填充表单
```

### 3. 数据录入流程
```
选择监测类型 → 填写表单/AI识别 → 自动计算密度 → 提交保存
```

### 4. 报告生成流程
```
选择报告类型 → 选择时间范围 → 后端生成 → 返回Markdown内容
```
