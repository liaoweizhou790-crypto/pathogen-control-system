# 猴痘消毒方案更新对比报告

**更新日期：** 2026年3月19日  
**数据来源：** 《猴痘防控方案》（中国疾控中心，2024年版）

## 一、防控方案要求的有效消毒剂

根据《猴痘防控方案》原文，对猴痘病毒有效的消毒剂包括：

| 序号 | 消毒剂名称 | 原文出处 | 系统是否包含 |
|:---|:---|:---|:---:|
| 1 | **75%乙醇** | 第2页（病原学特征） | ✅ |
| 2 | **含氯消毒剂** | 第2页（病原学特征） | ✅ |
| 3 | **过氧乙酸** | 第2页（病原学特征） | ✅ |
| 4 | **过氧化氢** | 第2页（病原学特征） | ✅ |
| 5 | **次氯酸钠** | 第2页（病原学特征） | ✅（属于含氯消毒剂） |
| 6 | **戊二醛** | 第2页（病原学特征） | ✅ |
| 7 | **邻苯二甲醛** | 第2页（病原学特征） | ✅ |
| 8 | **季铵盐类消毒剂** | 第16页（附件2） | ✅ |
| 9 | **热力消毒** | 第16页（附件2） | ✅ |
| 10 | **紫外线** | 第2页（病原学特征） | ✅ |

## 二、消毒方案更新对比

### 更新前（原系统数据）

| 消毒对象 | 原方案内容 |
|---------|-----------|
| 环境表面 | 含氯消毒剂 1000mg/L，60分钟 |
| 织物 | 含氯消毒剂 1000mg/L，60分钟 |
| 皮肤 | 碘伏 0.5% |
| 医疗器械 | 无明确方案 |
| 空气 | 无明确方案 |
| 餐饮具 | 无 |

### 更新后（按防控方案完善）

| 消毒对象 | 新方案内容 | 数据来源 |
|---------|-----------|---------|
| **环境表面** | **含氯消毒剂/过氧乙酸** 1000mg/L，60分钟<br>注：含氯消毒剂或过氧乙酸均可有效灭活病毒 | 第2页、第9页 |
| **织物** | **含氯消毒剂/季铵盐类/热力消毒**<br>注：单独洗涤和消毒，可选择热力消毒、含氯消毒剂或季铵盐类消毒剂浸泡 | 第16页附件2 |
| **皮肤/手** | **75%乙醇** 75%，擦拭消毒<br>注：接触病例后立即消毒 | 第2页 |
| **餐饮具** | **热力消毒/含氯消毒剂/季铵盐类**<br>注：餐具首选热力消毒，或含氯消毒剂、季铵盐类消毒剂浸泡 | 第16页附件2 |
| **医疗器械** | **戊二醛/邻苯二甲醛**<br>注：医疗器械专用，按《医疗机构消毒技术规范》执行 | 第10页 |
| **空气** | **紫外线/过氧化氢**<br>注：空气消毒可用紫外线照射或过氧化氢喷雾 | 第2页、第10页 |

## 三、关键更新说明

### 1. 皮肤消毒变更 ⚠️

| 项目 | 更新前 | 更新后 |
|------|--------|--------|
| **消毒剂** | 碘伏 0.5% | **75%乙醇** |
| **依据** | 原系统默认 | 《猴痘防控方案》第2页明确列出75%乙醇可有效灭活病毒 |

**说明：** 75%乙醇是防控方案明确列出的有效消毒剂，且皮肤消毒更常用乙醇。

### 2. 环境消毒补充 ✅

| 项目 | 更新前 | 更新后 |

| **依据** | 原系统默认 | 《猴痘防控方案》第2页明确列出过氧乙酸可有效灭活病毒 |

### 3. 织物消毒完善 ✅

| **依据** | 原系统默认 | 《猴痘防控方案》第16页附件2 |
