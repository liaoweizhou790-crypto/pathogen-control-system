# 病媒监测AI辅助数据录入方案

## 1. 方案概述

### 1.1 目标
通过AI技术实现**纸质原始记录的自动识别**和**数据的智能提取**，减少人工录入工作量，提高数据准确性和录入效率。

### 1.2 应用场景
- **现场纸质记录拍照识别**：监测人员拍摄纸质记录表，AI自动识别并填充到系统
- **病媒生物图像识别**：拍摄病媒生物照片，AI自动识别种类和数量
- **语音录入辅助**：现场通过语音快速录入数据
- **智能表单推荐**：根据历史数据和环境信息，智能推荐可能的监测结果

---

## 2. AI技术应用方案

### 2.1 纸质记录OCR识别（核心功能）

#### 技术方案
```
┌─────────────────────────────────────────────────────┐
│                 纸质记录AI识别流程                     │
├─────────────────────────────────────────────────────┤
│  1. 拍照 → 2. 图像预处理 → 3. OCR识别 → 4. 结构化    │
│       ↓           ↓            ↓           ↓        │
│    移动端      去噪/纠偏    文字识别    数据提取      │
│    拍摄       表格检测     手写识别    字段映射      │
│                                            ↓        │
│                                       人工确认      │
└─────────────────────────────────────────────────────┘
```

#### 技术选型
| 技术 | 方案 | 说明 |
|------|------|------|
| OCR引擎 | **PaddleOCR** / **百度OCR** / **腾讯OCR** | 支持手写体识别 |
| 表格识别 | Table Detection + OCR | 识别表格结构 |
| 手写识别 | Handwriting Recognition | 识别手写数字和文字 |
| 本地/云端 | 混合模式 | 简单识别本地，复杂识别云端 |

#### 支持的记录类型
- ✅ 蚊密度监测原始记录表（诱蚊灯法）
- ✅ 蝇密度监测原始记录表（捕蝇笼法）
- ✅ 蟑密度监测原始记录表（粘蟑法）
- ✅ 鼠密度监测原始记录表（粘捕法）
- ✅ 蜱虫监测原始记录表
- ✅ 布雷图指数监测记录表
- ✅ 双层叠帐法监测记录表
- ✅ 诱蚊诱卵器监测记录表

### 2.2 病媒生物图像识别

#### 功能设计
```
拍照识别功能：
┌─────────────────────────────────────┐
│  📸 拍摄病媒生物照片                  │
│         ↓                            │
│  🤖 AI智能识别                        │
│   - 蚊种识别（按属/种）                │
│   - 蝇种识别                          │
│   - 蟑螂识别（成蝇/若虫）              │
│   - 鼠种识别                          │
│   - 蜱虫识别                          │
│         ↓                            │
│  📊 自动统计数量 + 生成记录            │
└─────────────────────────────────────┘
```

#### 技术方案
- **预训练模型**：使用YOLOv8 / ResNet进行目标检测和分类
- **训练数据**：利用现有标本照片进行模型训练
- **分类层级**：
  - 蚊：按蚊属、库蚊属、伊蚊属等
  - 蝇：家蝇、大头金蝇、丝光绿蝇等
  - 蟑：德国小蠊、美洲大蠊、澳洲大蠊等
  - 鼠：小家鼠、褐家鼠、黄胸鼠等
  - 蜱：硬蜱、软蜱分类

## 5. 实施建议

### 5.1 分阶段实施

| 阶段 | 功能 | 技术方案 | 预期效果 |
|------|------|---------|---------|
| **第一阶段** | 纸质记录OCR识别 | 云端OCR API（百度/腾讯） | 识别率80%+ |
| **第二阶段** | 病媒生物图像识别 | 训练YOLOv8模型 | 识别常见种类 |
| **第三阶段** | 语音录入 | 科大讯飞API | 支持普通话/方言 |
| **第四阶段** | 本地AI识别 | 部署本地模型 | 离线可用 |

### 5.2 训练数据采集

为提高OCR识别准确率，需要收集以下训练数据：

1. **纸质记录表扫描件**（各类型各100张以上）
2. **病媒生物标本照片**（每种类别50张以上）
3. **手写数字样本**（0-9各100个样本）

### 5.3 识别准确率保障

| 措施 | 说明 |
|------|------|
| **置信度阈值** | 低于90%的字段标记为"需确认" |
| **人工复核** | 高置信度自动通过，低置信度人工确认 |
| **历史学习** | 根据人工修正结果持续优化模型 |
| **模板匹配** | 针对不同记录表类型使用专门模板 |
